🏦 Bank Management System (Excel VBA Project)
📌 Project Overview

The Bank Management System is a VBA (Visual Basic for Applications) based project developed using Microsoft Excel. It simulates basic banking operations such as account management, ATM functions, transactions, interest calculation, login security, and mini statements.

This project is designed for learning automation in Excel using VBA and demonstrates how banking operations can be handled in a simplified desktop environment.

🚀 Features

🔐 1. Login System
Secure user login using username and password
Prevents unauthorized access

🏧 2. ATM Module
Cash withdrawal
Balance inquiry
PIN verification

🧾 3. Account Management
Create new account
Update account details
View account information

💸 4. Transaction Module
Deposit money
Withdraw money
Transfer between accounts
Transaction history tracking

📊 5. Mini Statement
Displays recent transactions
Shows debit and credit history
Account-wise summary

💰 6. Interest Calculation
Calculates interest based on account balance
Supports simple interest logic (customizable)
🛠️ Technologies Used
Microsoft Excel
VBA (Visual Basic for Applications)
UserForms (for UI design)

📂 Project Structure
Login Form → User authentication screen
ATM Module → ATM operations
Account Sheet → Stores account details
Transaction Sheet → Stores all transactions
Mini Statement → Shows recent activity
Interest Module → Calculates interest
